* About
Implementation of [[https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life][Conway's Game of Life]] in Rust.

* Quick Start
#+begin_src console
$ git clone https://github.com/shoumodip/gol-rs
$ cd gol-rs
$ cargo install --path .
$ gol-rs examples/glider.gol
#+end_src
